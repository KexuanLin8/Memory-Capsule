# 作品说明文档：Memory Capsule (记忆胶囊)

## 1. 作品名称
**Memory Capsule (记忆胶囊) —— 您的 3D 沉浸式记忆宇宙**

## 2. 作品简介
**Memory Capsule** 是一个融合了 **AIGC (人工智能生成内容)**、**WebGL 3D 可视化** 与 **计算机视觉交互** 技术的数字化记忆存储平台。

它打破了传统相册和日记的二维限制，利用先进的 AI 算法（Hunyuan 3D & Vector Engine），将用户上传的平面照片或输入的文字描述瞬间转化为生动的 **3D 立体模型**。这些模型被珍藏在一个浩瀚的“记忆宇宙”中，用户可以像翻阅历史长河一样，通过底部的时间轴穿越于 1980 年至 2030 年之间。

更独特的是，作品集成了 **MediaPipe 手势识别** 技术，允许用户隔空挥手来旋转记忆，或双手合璧来放大细节，带来仿佛置身于科幻电影般的沉浸式交互体验。

## 3. 目标人群
*   **情感记录者**：热衷于记录生活、追求生活仪式感，希望以更独特方式保存珍贵时刻（如婚礼、毕业、宠物成长）的年轻人及家庭用户。
*   **科技尝鲜者**：对 AI 作画、3D 建模、体感交互等前沿 Web 技术感兴趣，渴望体验新奇互联网产品的极客群体。
*   **回忆疗愈需求者**：希望重温美好往事，寻找情感寄托，通过沉浸式回顾获得心灵慰藉的人群。
*   **创意创作者**：需要快速将灵感或概念转化为 3D 原型进行展示的设计师或艺术家。

## 4. 解决什么问题
1.  **解决“记忆扁平化”痛点**：传统的照片和文字是静止的、平面的。本作品通过 AI 3D 重建技术，让记忆“站”起来，还原记忆发生时的空间感和立体感，提供比 2D 照片更丰富的信息量和更强烈的临场感。
2.  **打破“交互枯燥”的局限**：普通电子相册仅能滑动浏览。本作品引入**手势控制（Gesture Control）**，用户无需触碰屏幕，只需挥手即可全方位把玩记忆模型，增加了互动的趣味性和未来感。
3.  **降低“3D 创作”门槛**：以往制作 3D 模型需要掌握复杂的建模软件（如 Blender）。本作品利用 AI 代理了这一过程，用户只需“说”出记忆（Prompt）或“给”张照片，几分钟内即可生成高质量 3D 模型，让普通人也能拥有自己的 3D 资产。
4.  **强化“情感连接”**：通过精心设计的 UI（星空背景、情感化打字机标语、时光轴），营造出一种神圣、深邃的氛围，帮助用户在快节奏的数字生活中慢下来，与自己的过去建立更深层的情感连接。
